## 3. HTML 태그들
***
### HTML
1. HTML 기본 문법
- 콘텐츠를 가지는 태그 `<div> 콘텐츠 </div>`
- 콘텐츠를 가지지 않는 태그  `<br />`
- 기본 작성법 `<태그 이름>`
2. 속성과 값
- `<div 속성="값"> Content </div>` 으로 적용 
3. HTML 기본 문서
```html
<!DOCTYPE>
<html lang="ko">
    <head>
      <title>문서 제목</title>
    </head>
    <body>
        안녕하세요!
    </body>
</html>
```
